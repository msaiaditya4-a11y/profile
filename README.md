## 📌 Project Overview
This is a **responsive Personal Portfolio Website** built as part of my **Online Full Stack Development Internship – Task 1**.

It showcases my skills, projects, and contact details in a clean and modern UI using HTML, CSS, and JavaScript.

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Font Awesome Icons  

---

## ✨ Features

### 💻 Sections
- Home / Hero Section with introduction  
- About Me section  
- Skills section with progress bars  
- Projects showcase  
- Contact form with social links  

### 🎨 UI Features
- Fully responsive design 📱💻  
- Smooth navigation  
- Modern UI design  
- Animated skill bars  
- File upload option for profile image  
- Social media integration
